# WM-SQLITE

WM-SQLITE is a wrapper around the SQLITE3 database library for Common
Lisp.

## Components

WM-SQLITE contains serveral components to provide the interface to the
SQLITE3 library.

 * Databases
 * Transactions
 * Statements
 * Statement cache
 * Query streams
 * Blob streams
 * An SQL templating system
 * Persistent objects
 * Caching of persistent objects
